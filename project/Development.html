Welcome to the EduGame development website
Here we provide all information for those interested in contributing to the game's development on this website.

Available Information
On this page, you can find the progress of the development process includes: code status, issues that need to be fixed and new beta versions of the game that need developers' contributions.

Beta Version (Version 0.2)
Until now, all games are available in the English language. Therefore, we are working on supporting additional languages. Moreover, we are facing difficulties in finding sound libraries that support other languages, thus we try to developing jQuery libraries in multiple languages. Besides, we are developing beta versions of learning colors game, learning letters game and learning math operations game, so we would invite who has a willingness to test new games and bug fixes before they will be added to the next release to contact us on developers@EduGame.com.
