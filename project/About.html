EduGame is a website of educational games designed with educational and entertainment purposes for little children. It contains four types of games: Learn Numbers, Learn Time, Memory Cards and Puzzle Game. All four games can be used in the educational environments with the aim to assist children to learn about certain subjects as they play.

EduGame is developed by: Nourah Al-Mutlaq & Alhanoof Al-Ajmi.
